# @chimeric/rtk-query

RTK Query utilities for chimeric interfaces. Provides dual idiomatic (function call) and reactive (React hook) interfaces for RTK Query endpoints.

## Installation

```bash
npm install @chimeric/rtk-query @chimeric/core @reduxjs/toolkit react-redux
```

## Usage

See the [chimeric documentation](https://dataquail.github.io/chimeric) for full usage details.
