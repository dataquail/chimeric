For usage details, see [AsyncReducer](/chimeric/api/async-reducer/).

AsyncReducer utilities are provided by `@chimeric/rtk-query`:

```typescript
import {
  ChimericAsyncReducer,
  IdiomaticAsyncReducer,
  ReactiveAsyncReducer,
} from '@chimeric/rtk-query';
```

Instantiation and usage are identical to the shared [Reducers](/chimeric/api/reducers/) documentation.
